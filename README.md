# EDNA Images

This repository can be used to store non-sensitive images that can be referenced in BI platforms like Sigma, Tableau, etc.

## Adding images
To add images, you will need:
* A Github Account
* Access to the Extreme Networks Github organisation
* Write access on this repository

To add images:
* On the main branch, click `Add File` and then `Upload Images`
* Choose your images and upload them

## Referencing images in BI tools
* To obtain the URL to reference your image, open the image on Github, right-click the image and copy the image link
